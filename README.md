## 🧩 プロジェクト概要

Spring Bootを使用して開発した料理が好き人に向けるブログウェブサイトです。ユーザーはログインして記事を投稿・編集・削除でき、他ユーザーの投稿を見るすことも可能です。

---
## ⚙️ 使用技術

- バックエンド：Java 17 / Spring Boot / JPA
- フロントエンド：HTML / CSS / JavaScript / Thymeleaf
- データベース：PostgreSQL
- ビルド・管理：GitHub
